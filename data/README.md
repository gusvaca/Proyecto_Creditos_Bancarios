# Carpeta de Datos

Esta carpeta contiene los datos utilizados en el proyecto.

## Archivos

- `creditos_bancarios.csv` - Dataset principal con 5000 registros de solicitudes de crédito

## Variables del Dataset

- Edad
- Género
- Ciudad
- Ingresos Mensuales
- Gastos Mensuales
- Score Crediticio
- Antigüedad Laboral
- Mora Histórica
- Capacidad de Pago
- Crédito Aprobado
