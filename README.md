# Proyecto de Créditos Bancarios

Proyecto de Evaluación Data Sintética Analítica y Big Data - Universidad Hemisferios - Gustavo Vaca

## Descripción
Este proyecto incluye análisis de datos de créditos bancarios.

## Estructura del Proyecto
- `data/` - Archivos de datos
- `notebooks/` - Análisis y notebooks
- `scripts/` - Scripts de procesamiento
- `results/` - Resultados y reportes
