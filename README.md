# Predicción de Aprobación de Créditos
https://github.com/gusvaca/Proyecto_Creditos_Bancarios.git
## Descripción

Proyecto desarrollado para la asignatura Big Data, Analytics & Data Science.

El objetivo es analizar solicitudes de crédito bancario mediante técnicas de análisis exploratorio de datos (EDA) y Machine Learning para predecir la aprobación de créditos.

## Objetivos

- Analizar la calidad y estructura del dataset.
- Realizar análisis exploratorio de datos.
- Construir visualizaciones para identificar patrones.
- Implementar un modelo Random Forest.
- Evaluar el desempeño mediante métricas de clasificación.

## Tecnologías Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-Learn
- Jupyter Notebook

## Dataset

El dataset contiene 5000 registros de solicitudes de crédito bancario e incluye variables demográficas, financieras y crediticias.

Principales variables:

- Edad
- Género
- Ciudad
- Ingresos Mensuales
- Gastos Mensuales
- Score Crediticio
- Antigüedad Laboral
- Mora Histórica
- Capacidad de Pago
- Crédito Aprobado

## Metodología

1. Carga y exploración del dataset.
2. Limpieza y transformación de datos.
3. Análisis exploratorio de datos (EDA).
4. Visualización de información.
5. Construcción del modelo predictivo.
6. Evaluación de métricas.
